《浪潮之巅》 Reading Notes

Posted on 2017-05-16   |   In Tech   |   | 阅读量 次

这本《浪潮之巅》断断续续看了半年多，它主要分析了近一个世纪中主要科技公司的兴衰历史和成功规律，它们就像是这个时代一个接一个的浪潮。作者吴军毕业于清华和Johns Hopkins，并先后就职于Google和腾讯。看完这本书可以对科技行业有一个整体的认识，能够大致判断一个科技公司是否有前景。一个残酷而又美好的事实是：和最基本的自然规律一样，行业更替在不断进行，不能适应市场变化的公司将最终被淘汰，新的公司会登上历史舞台。

信息产业规律

摩尔定律: 集成电路的集成度每两年会翻一番
安迪比尔定律: 比尔(比尔盖茨)要拿走安迪(原英特尔CEO)所给的，用户不断更新硬件
反摩尔定律: 同样的产品每两年价格降一半
70-20-10市场份额
诺威格定律: 当市场份额饱和时需要扩展和转型，找到新增长点
基因决定定律

Silicon Valley

创业公司需要好的创始人，商业模式，外部环境
存活率很低
灵魂是创新
其他模仿硅谷的地方给人才设置门槛很难成功
好的商业模式: 增加自动化，减少人力成本

AT&T

贝尔实验室
1984年司法部拆分成七家公司，主营业务由原来的通话业务转向了新兴的网络和移动通信
1995年公司高管在华尔街推动下进行自我拆分从而衰落
专门从事电信业务的AT&T公司、从事设备制造业务的朗讯和从事计算机业务的NCR

IBM

创始人Thomas J. Watson
制表机，兼容机，服务器
主要服务对象为政府和大型企业

Apple

富于创新的消费电子公司
图形化界面操作系统被盖茨看到了，与微软打官司，最终和解
乔布斯被踢出，收购皮克斯，重返苹果
Macintosh纵向封闭，Windows横向开放
iPod, iPhone, AppleTV

Intel

创始人Gordon Moore
大量投入研发芯片
坚持CISC
移动端没有市场

Microsoft

利用DOS拖延IBM直到开发出Windows
捆绑IE搞垮Netscape，强调IE是Windows的功能而非软件

Cisco

路由器
支持内部创业或离职创业之后收购
华为成本优势占领中低端市场

Yahoo

创始人杨致远
开放，免费
卖广告流量
本身可以做好媒体广告，却要和Google拼搜索广告技术

Hewlett-Packard (HP)

斯坦福大学出租土地成立斯坦福工业园
拆出科学、医疗仪器部门，发展计算机与打印机
消费家电公司

Motorola

无线通信设备不如诺基亚，DSP不如TI，CPU不如Intel
铱星计划：66颗近地卫星不需要地面基站
注重产品质量

Sun

开源Solaris提供服务
Java, JSP与微软ASP竞争
被Oracle收购

Novell

微机局域网操作系统，输给Windows NT

Venture Capital

世界经济发展 -> 股票呈上涨趋势
科技发展 -> 风险投资存在
每次投资成立一个有限责任公司(特拉华州，开曼群岛，巴哈马)
天使投资: 小型风险投资
知识产权，革新性技术，商业模式，进入壁垒
储备职业经理人
Sand Hill Road: KPCB, Sequoia Capital, NEA, Mayfield, IDG, Softbank

Stanford University

经费大多来自工业界(注重应用研究)，东部名校经费大多来自政府(注重理论研究)
开放，职业教育，多样性，培养全才，创业气氛

Google

把互联网内容免费且方便地送给用户
不作恶，搜索结果公正且没有人为干预
搜索广告数量少，价格高
针对搜索关键词和网页内容投放高度相关的广告
早期员工都是精英
学习微软只做操作系统

Nokia

由芬兰木工厂转型
支持欧洲GSM标准
抓住数字通信的机遇，注重功能、方便、新颖

3M

胶带，软盘，液晶显示器膜
允许员工15%的工作时间用于自由创新
卖掉还在盈利但是前景不好的部门
面向大众的产品比面向企业的产品营收更稳定

GE(General Electric)

爱迪生创立
电力，能源，航空发动机，医疗，金融

Ebay

在线拍卖旧货转型为在线商店
收购Paypal，后来分拆
甩手掌柜不管交易

Amazon

在线书店转型为在线商店
作为中间人保障交易顺利

Dell

制造业环节: 产品设计，原料采购，仓储运输，加工制造，订单处理，销售
只负责订单处理和直销，其他全部外包

Web 2.0

UGC，开放，交互，不与用户竞争
难以盈利，流量价值不如搜索引擎

经济危机

经济危机也是优胜劣汰的机遇
资本从实体经济向虚拟经济转移
超前消费
资产阶级革命: 封建领主用特权交换财富

下一个风口

新能源，核能，电动汽车
生物制药

BAT

机遇: 移动互联网，云计算，互联网金融
百度: 过度依赖搜索广告，转型缓慢，押宝人工智能
阿里巴巴: 电商，支付，云计算
腾讯: 社交，游戏
阿里、腾讯的市值是百度的4倍
